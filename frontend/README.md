# Frontend

## Run locally

```
npm run serve
```

# Setup

Need:
* Node v16
* npm 8.5.0

Vue V2 not V3 

```
npm i @vue/cli@3.12.1
```

## Web Components

```
vue build --target wc --name community-tools-news src/components/News.vue
```
